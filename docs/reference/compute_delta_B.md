# Compute treatment effect on between-group inequality

Compute treatment effect on between-group inequality

## Usage

``` r
compute_delta_B(pi, mu0, beta, ystat)
```

## Arguments

- pi:

  Group shares among treated

- mu0:

  Baseline group means mu_j(0)

- beta:

  Treatment effects on mean

- ystat:

  "Var" or "CV2"

## Value

Scalar delta_B^D
